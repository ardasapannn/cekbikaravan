# ÇekBiKaravan Statik Site

Bu klasör `çekbikaravan.com` için ilk statik HTML altyapısını içerir.

## Önizleme

`index.html` dosyasını tarayıcıda açarak siteyi görebilirsin.

## Düzenlenecek Alanlar

- `index.html`: Metinler, telefon, WhatsApp ve e-posta bilgileri.
- `styles.css`: Renkler, boşluklar ve responsive tasarım.
- `assets/hero-caravan.png`: Ana sayfa karavan görseli.

## Yayına Alma

Statik hosting için tüm dosyaları alan adının bağlı olduğu hosting kök dizinine yüklemek yeterli:

- `index.html`
- `styles.css`
- `script.js`
- `assets/hero-caravan.png`
